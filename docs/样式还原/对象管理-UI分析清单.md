## 七、实现建议

### 7.1 组件划分与样式文件

- 布局壳层：`src/layout/console-layout.vue`
- 页面入口：
	- `src/views/workbench/index.vue`
	- `src/views/object-type-list/index.vue`
	- `src/views/object-type-create/index.vue`
- 创建页步骤子组件：
	- `step-source.vue`
	- `step-metadata.vue`
	- `step-attributes.vue`
	- `step-actions.vue`

### 7.2 实现时优先规则

- 优先保证布局与交互主流程一致，再处理图标和微视觉细节。
- 色彩和背景使用主题变量，避免新增硬编码主色。
- 表格与卡片布局尽量避免空白断层和元素截断。

### 7.3 常见错误规避

- 避免只实现 default 态，遗漏创建页步骤切换逻辑。
- 避免分页区仅保留页码，遗漏页容量与前往页控件。
- 避免把估算值写成“精确值”，必须保留证据等级。
