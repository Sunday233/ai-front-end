## 2. 布局 Map

1. 左侧固定导航栏：品牌、租户信息、分组菜单、计数徽标。
2. 顶部面包屑：返回首页 / 智能体管理 Matrix / 当前页面。
3. 内容区：工具栏 + 主体卡片区。

## 3. 页面级还原要点

## 3.1 工作台

1. 顶部工具栏包含：搜索输入、分支选择、查询重置、新建按钮。
2. 中间区为最近浏览对象卡片，3 列布局，卡片内包含标题、说明、标签。
3. 底部区为收藏对象类型，默认空态展示。

## 3.2 对象类型列表

1. 顶部标题区包含名称与数量徽标。
2. 工具区包含筛选、刷新、新建。
3. 主区为表格，列包含类型名称、状态、可见性、日期、操作。
4. 底部为分页操作区。

## 3.3 对象类型创建

1. 顶部 4 步进度条：选择数据源、元数据配置、属性配置、动作配置。
2. Step1：数据源模式二选一、数据集名称、保存位置。
3. Step2：对象类型名称、别名、描述、对象组复选。
4. Step3：主键/标题键与属性列表编辑。
5. Step4：动作类型复选与执行组选择。
6. 底部操作按钮：上一步 / 下一步 / 完成。

## 4. 样式与视觉基线

1. 主色使用蓝色体系，对齐 Ant Design 主色变量。
2. 主体背景为浅灰，卡片为白底并带轻量阴影。
3. 导航激活项采用浅蓝底与蓝色文字。
4. 状态通过圆点 + 文案表达，避免仅靠颜色。

## 5. 交互行为基线

1. 工具栏查询行为：输入关键字后刷新列表。
2. 创建流程行为：每步校验通过后可进入下一步。
3. 完成创建后返回列表页。
